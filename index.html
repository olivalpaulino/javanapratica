Do Zero ao Avançado

Aqui, você segue uma trilha de estudos do zero ao avançado. Comece pelo Java na Prática, onde você vai aprender os recursos básicos da linguagem, entender o conceito mais utilizado que é a Orientação a Objetos, e ainda vai ganhar uma imersão de Banco de Dados com MySQL, para aprender como tudo funciona na lógica de negócio.

O segundo passo, seria ir para os projetos de aprofundamento, que são Clínica Médica (Construindo uma Fila de Espera com Banco de Dados na Prática), e o Sistema de Lanchonente, onde utilizamos Java, MySQL e Janelas Gráficas para te dar uma experiência Full Stack. Neste último projeto, de lanchonente, você pode aprofundar a parte de criação de janelas gráficas com Swing, no curso isolado de Java Swing.

Aqui, você já está preparado para temas mais avançados, como o estudo de Java na Web, através do framework mais utilizado no mercado, que é o Spring Boot.

Por fim, para aqueles que desejam ter uma experiência web full stack, indo além do back-end web que veremos com Spring Boot usando Spring MVC, DataJPA e Security, com materiais complementares, você pode estudar a tecnologia Thymeleaf, que é o curso Java no Front-end, para criar os famosos templates Web com Spring Boot e Thymeleaf.

💡 Lembrando que também temos os cursos de inteligência artificial com foco na construção de uma nova fonte de renda, e criação do seu próprio negócio digital, trazendo temas de vendas, marketing digital, produção de conteúdo e muito mais.
